PROPERTIES OF xor:
xor is commutative and associative 

1. 0^1 = 1
2. 1^0 = 1
3. 0^0 = 0
4. 1^1 = 0

 xor of number with itself is 0
   a^a = 0

 xor of number with 0 is number itself
   a^0 = a  


 xor with 1 of number results in complement of number
   a^1 = ~a
